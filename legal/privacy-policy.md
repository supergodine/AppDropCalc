# POLÍTICA DE PRIVACIDADE - DROPCALC

**Última atualização: 2 de novembro de 2025**

## 1. INFORMAÇÕES GERAIS

A presente Política de Privacidade contém informações sobre coleta, uso, armazenamento, tratamento e proteção dos dados pessoais dos usuários do aplicativo **DropCalc**, com a finalidade de demonstrar absoluta transparência quanto ao assunto e esclarecer a todos interessados sobre os tipos de dados que são coletados, os motivos da coleta e a forma como os usuários podem gerenciar ou excluir as informações pessoais.

Esta Política de Privacidade aplica-se a todos os usuários do DropCalc e integra os Termos e Condições Gerais de Uso do aplicativo.

## 2. COMO RECOLHEMOS OS DADOS PESSOAIS

Os dados pessoais do usuário são recolhidos pela plataforma da seguinte forma:

### 2.1 Dados fornecidos pelo usuário:
- **Cadastro de conta:** nome, e-mail e senha
- **Login via Google:** nome, e-mail e foto de perfil (quando autorizado)
- **Configurações:** moeda padrão, país de origem
- **Cálculos:** dados inseridos nas calculadoras (valores de produtos, margens, etc.)

### 2.2 Dados coletados automaticamente:
- **Dados de dispositivo:** tipo de dispositivo, sistema operacional, versão do aplicativo
- **Dados de uso:** funcionalidades utilizadas, tempo de sessão, frequência de uso
- **Dados técnicos:** endereço IP, logs de acesso, dados de performance

## 3. QUAIS DADOS PESSOAIS RECOLHEMOS

### 3.1 Dados pessoais fornecidos pelo usuário:
- **Identificação:** nome completo, endereço de e-mail
- **Autenticação:** senha criptografada, tokens de acesso
- **Preferências:** configurações de idioma, moeda, país
- **Cálculos:** histórico de cálculos realizados (valores, produtos, margens)

### 3.2 Dados sensíveis:
O DropCalc **NÃO** coleta dados sensíveis como:
- Dados bancários ou cartão de crédito
- Documentos pessoais (CPF, RG, etc.)
- Dados biométricos
- Informações médicas

## 4. PARA QUE FINALIDADES UTILIZAMOS OS DADOS PESSOAIS

Os dados pessoais do usuário coletados e armazenados pelo DropCalc têm por finalidade:

### 4.1 Finalidades principais:
- **Prestação de serviços:** fornecer funcionalidades da calculadora
- **Gerenciamento de conta:** autenticação e personalização
- **Histórico:** manter registro dos cálculos realizados
- **Suporte técnico:** resolver problemas e fornecer assistência

### 4.2 Finalidades secundárias:
- **Melhorias:** análise de uso para aprimorar o aplicativo
- **Comunicação:** envio de atualizações e informações importantes
- **Segurança:** prevenção de fraudes e atividades maliciosas

## 5. PRAZO DE CONSERVAÇÃO DOS DADOS PESSOAIS

Os dados pessoais do usuário são conservados pelo período necessário para a prestação do serviço ou cumprimento das finalidades previstas neste documento, conforme o disposto no inciso I do artigo 15 da Lei 13.709/18.

### 5.1 Prazos específicos:
- **Dados de conta:** enquanto a conta estiver ativa + 5 anos após exclusão
- **Histórico de cálculos:** 2 anos após o último acesso
- **Logs de acesso:** 6 meses
- **Dados de marketing:** até a revogação do consentimento

## 6. COMPARTILHAMENTO DOS DADOS

### 6.1 O DropCalc compartilha dados pessoais com:
- **Provedores de autenticação:** Google (quando usar login social)
- **Serviços de infraestrutura:** provedores de hosting e CDN
- **Analytics:** Google Analytics (dados anonimizados)
- **Autoridades:** quando exigido por lei ou ordem judicial

### 6.2 O DropCalc NÃO compartilha dados com:
- Empresas de marketing não autorizadas
- Terceiros para fins comerciais
- Redes sociais (exceto login autorizado)

## 7. TRANSFERÊNCIA INTERNACIONAL DE DADOS

Alguns dados podem ser transferidos para servidores localizados fora do Brasil, especificamente:

- **Google Cloud Platform:** dados de autenticação via Google
- **Cloudflare:** dados de CDN e segurança
- **AWS:** backups e infraestrutura

Todas as transferências seguem adequadas garantias de proteção conforme a LGPD.

## 8. SEUS DIREITOS

De acordo com a Lei Geral de Proteção de Dados (LGPD), você possui os seguintes direitos:

### 8.1 Direitos do titular:
- **Confirmação da existência de tratamento**
- **Acesso aos dados pessoais**
- **Correção de dados incompletos, inexatos ou desatualizados**
- **Anonimização, bloqueio ou eliminação de dados desnecessários**
- **Portabilidade dos dados a outro fornecedor**
- **Eliminação dos dados tratados com consentimento**
- **Informação sobre compartilhamento de dados**
- **Revogação do consentimento**

### 8.2 Como exercer seus direitos:
Entre em contato conosco através do e-mail: **privacidade@dropcalc.app**

## 9. SEGURANÇA DOS DADOS

O DropCalc emprega medidas técnicas e organizativas aptas a proteger os dados pessoais:

### 9.1 Medidas técnicas:
- **Criptografia:** dados sensíveis são criptografados
- **HTTPS:** todas as comunicações são protegidas
- **Autenticação:** senhas são hasheadas com bcrypt
- **Backups:** backups regulares e seguros

### 9.2 Medidas organizativas:
- **Acesso limitado:** apenas funcionários autorizados
- **Treinamento:** equipe treinada em proteção de dados
- **Auditorias:** revisões regulares de segurança

## 10. COOKIES E TECNOLOGIAS SIMILARES

O DropCalc utiliza cookies e tecnologias similares para:

### 10.1 Tipos de cookies:
- **Essenciais:** necessários para funcionamento básico
- **Funcionais:** melhoram a experiência do usuário
- **Analytics:** coletam dados de uso anonimizados

### 10.2 Gerenciamento:
Você pode gerenciar cookies através das configurações do seu navegador.
